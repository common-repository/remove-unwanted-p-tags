== Description ==
It will remove auto generated p tags in wordpress content. 

== Installation ==
Steps to follow.
1. Click on \"Add New\" button in \"Plugins\" menu.
2. Search for \"Remove Unwanted P tags\" plugin.
3. Click on the \"Install\" button & then after \"Active\".
4. You\'re ready to use plugin.
